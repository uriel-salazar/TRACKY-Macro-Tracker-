# How It's Made:
## Tech used :
- Python 3
- Command-line interface (CLI)

- Requests library (API communication)
- Pandas

# Getting Started
## Prerequisites :
- Python 3.10 or higher
- pip (Python package manager)

## Installation:
*1*. **Clone repository**
```bash
git clone https://github.com/uriel-salazar/TRACKY.git
```
*2*. **Enter the project folder :**
```bash
pip install -r requirements.txt
```
*3*. **Running :**
```bash
python main.py
```
